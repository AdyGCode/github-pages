Step 1: Create GitHub Pages Repo

Open Github.

Create a new repository YOURGITHUB.github.io, replacing YOURGITHUB with your GitHub account name.

For example adygcode.github.io

Make sure the Repository is public and that you initialise the repo with a ReadMe.md file.

Clone the repository to your local machine using either WebStorm, or the command line, or another suitable IDE.

Open your Editor and create a new index.html file

In the basic file add (between the <body> ... </body> tags) <h1>Placeholder for YOURGITHUB</h1>.

Commit the new file with 'Initial Commit'

Delete the ReadMe.md file

Commit the change to your repository.

Push the chenges to the origin.

You hsould now be able to visit https://YOURGITHUB.github.io
